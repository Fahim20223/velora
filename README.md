# 👕 Velora

A modern fashion e-commerce frontend built with **Next.js**, **React**, and **Tailwind CSS**.

## 🌐 Live Demo

👉 https://velora-xi-sooty.vercel.app/

## ✨ Features

- Responsive Design
- Product Listing
- Product Details Page
- Dynamic Routing
- Shopping Cart
- Cart Persistence with Local Storage
- Toast Notifications
- Modern UI

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS
- Context API
- React Toastify
- React Icons

## 🚀 Run Locally

```bash
git clone https://github.com/Fahim20223/velora.git

cd velora

npm install

npm run dev
```
